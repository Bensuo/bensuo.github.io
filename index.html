Neon Knights

Neon Knights was developed as part of a small team for the Microsoft Imagine Cup 2016. The game was developed in Monogame/C# and I served as a programmer on the project, mostly responsible for gameplay programming. The game is a multiplayer side-scrolling brawler/party game where each round the game mode and level are randomised along with an additional random gameplay modifier (such as meteors striking from the sky).

The team for this project was myself (programming), Stuart Adams (programming and additional art) and Alex McCauley (art and animation).

  • Date: Spring 2016

Fluid Simulation

A grid based fluid simulation developed as part of a group for our Interactive Physical Modelling module. The simulation is largely based on Jos Stam's methods described in the paper "Real-Time Fluid Dynamics for Games".

The team for this project was myself, Anthony Palm and Thomas Mackrell.

3D First/Third Person Shooter

Developed as part of a group project for the Real Time 3D Graphics module at university. The game was developed in C++ using SDL and GLEW/OpenGL. I was responsible for the terrain generation, collisions (AABBs, terrain and raycast), rendering and AI as well as some shader work.

The team for this project was myself, Stuart Adams and Alistair Buchanan.

  • Date: April 2017

RedMetal / TinyBox Engine

Developed for the 3rd year Game Project module at UWS. This involved the creation of the TinyBox engine which uses a Unity styled Entity-Component design, as well as the Wipeout inspired anti-gravity racing game RedMetal to showcase the engine. The Engine aims to present a high level interface for game creation while still allowing a good amount of flexibility when working with the systems. I was primarily responsible for the high-level graphics interface and rendering pipeline, audio, particle systems and created many of the game's systems.

The team for this project was myself, Stuart Adams and Alistair Buchanan.

  • TinyBox Source: Here
  • Date: April 2018

Geometry Shader Fur

Developed for my 3rd year Advanced Graphics Programming module at UWS. Fur rendered on a 3D Model using Shells generated in OpenGL geometry shaders. The demo also features dynamic control of properties such as gravity, wind force, fur length and shell count.

  • Date: December 2017
